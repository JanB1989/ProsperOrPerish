# Paradox bug report — file checklist

Use when submitting a **minimal** EU5 crash with Prosper or Perish (or any mod).

- [ ] `exception.txt` from crash folder
- [ ] `meta.yml` (version, mods, GPU, render API)
- [ ] Full `logs/error.log` from crash folder (not only `Documents/.../logs/error.log` tail)
- [ ] `pdx_settings.json` from crash folder (optional)
- [ ] Save file **or** exact bookmark + start date + steps
- [ ] Mod version: Steam id or zip + `metadata.json` / `.metadata/metadata.json`
- [ ] Short numbered repro (from [eu5_pp_crash_investigation.md](./eu5_pp_crash_investigation.md) Phase 0 matrix)
- [ ] Note: DLSS on/off and Vulkan vs D3D12 tested

Game build identifiers are in `meta.yml` (`SCMCommit`, `AppVersion`).
